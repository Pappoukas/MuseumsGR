# 🏛️ Ανάλυση Επισκεψιμότητας Ελληνικών Μουσείων (1998-2025)

Αυτό το project αποτελεί ένα διαδραστικό εργαλείο οπτικοποίησης δεδομένων (Data Visualization Dashboard) για την κίνηση των μουσείων στην Ελλάδα, βασισμένο στα επίσημα στοιχεία της **ΕΛΣΤΑΤ**.

## 🚀 Δυνατότητες
- **Διαδραστικά Γραφήματα:** Παρακολούθηση τάσεων από το 1998 έως και τον Οκτώβριο του 2025.
- **Φιλτράρισμα:** Επιλογή συγκεκριμένων μουσείων ή χρονικών περιόδων.
- **Περιγραφική Στατιστική:** Αυτόματος υπολογισμός μέσων όρων και συνολικών επισκεπτών.
- **Data Export:** Δυνατότητα λήψης των φιλτραρισμένων δεδομένων σε μορφή CSV.

## 🛠️ Τεχνολογίες
- **Python 3.x**
- **Streamlit** (Web Interface)
- **Pandas** (Data Manipulation)
- **Plotly** (Interactive Charts)

## 📊 Πηγή Δεδομένων
Τα δεδομένα προέρχονται από τα μηνιαία δελτία της **Ελληνικής Στατιστικής Αρχής (ΕΛΣΤΑΤ)**.

## 💻 Πώς να το τρέξετε τοπικά
1. Εγκαταστήστε τις βιβλιοθήκες: `pip install -r requirements.txt`
2. Τρέξτε την εφαρμογή: `streamlit run app.py`
