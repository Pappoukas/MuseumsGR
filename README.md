# MuseumsGR
Διαδραστική ανάλυση επισκεψιμότητας 201 Ελληνικών Μουσείων (1998-2025). Data visualization &amp; στατιστικά στοιχεία βασισμένα σε δεδομένα της ΕΛΣΤΑΤ. Ένα εργαλείο λήψης αποφάσεων για το πολιτιστικό marketing, φτιαγμένο με Python &amp; Streamlit.

Σκοπός: Η μετατροπή πρωτογενών δεδομένων σε actionable insights για τον τουρισμό.

Τεχνολογίες: Python (Pandas, Plotly), Streamlit Cloud.

Πηγή: Επίσημα μηνιαία δελτία Ελληνικής Στατιστικής Αρχής.
